第 1 章 Elegant LaTeX 系列模板介绍

Elegant LaTeX 项目组致力于打造一系列美观、优雅、简便的模板方便用户使用。目前由 ElegantNote， ElegantBook， ElegantPaper 组成，分别用于排版笔记，书籍和工作论文。如果你在使用本模板，推荐最新版本！最新正式版下载地址： Github。本文将介绍本模板的一些设置内容以及基本使用方法。如果您有其他问题，建议或者意见，欢迎在 Github 上给我们提交 issues 或者邮件1联系我们。最近我们新建了一个 QQ 用户交流群（Q 群：692108391），欢迎加入。

1.1 ElegantBook 更新说明

在这几年间，我们收到了很多用户的反馈，主要的问题涉及到字体安装，编码支持，定理浮动，定理跨页，交叉引用等等。我们思前想后，原先让用户安装字体以追求视觉上的美观并不完美，用户陷入了巨大的麻烦，这违背了我们的模板初衷。因此我们在新版中删除了这部分，用户无需安装任何字体。ElegantBook 3.x 版本更新的主要内容有：

  1. 删除了自定义字体设置，改用 ctex 宏包支持中文；
  2. 模板拆分为中英文模式（lang=cn/en）；
  3. 支持 PDFLaTeX 与 XeLaTeX；
  4. 使用 tcolorbox 宏包改写定理类环境，可跨页；
  5. 定理类环境名字更新，修复定理环境交叉引用；
  6. 更新颜色主题，增加灰色和黑色主题；
  7. 颜色名字更新，统一链接颜色；
  8. 重新绘制 Elegant LaTeX 的 Logo；
  9. 更新封面与装饰物，删除水印；
  10. 修正附录相关内容；
  11. 增加代码高亮；
  12. 美化列表环境；
  13. 增加简化模式；
  14. 增加 base 隐藏选项。

由于新版本进行了重构，3.x 版本并不兼容 2.x 版本，如果你想把 2.x 版本的文件转为 3.x 版本，请查看 跨版本转换。今后不再回答关于 2.x 版本的问题。
